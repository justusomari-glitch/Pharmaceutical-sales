# Pharmaceutical-sales
The project focuses on analysing pharmaceutical sales,inventory and customer transactions using sql. The dataset simulates real world pharmacy environment and includes multiple related tables such as products,customers,sales and inventory. The goal of the project was to demonstrate practiical skills used in data analytics including cleaning,joining multiple tables,aggregations and business driven analysis

key objectives of the project are
1.Annalyse sales perfomances across products and time period
2.Track inventory levels and identify low stock items
3.Understand customer purchasing patterns

Skills and concepts applied
1.SQL JOINS
2.Data cleaning
3.Group-by and aggregate functions
Business focused analysis
